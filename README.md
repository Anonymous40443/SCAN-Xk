## 🚀 Funcionalidades de Elite

### 🔍 Reconhecimento (Nmap Mega Module)
* **15+ Tipos de Scan:** Desde Stealth SYN até detecção agressiva de SO e vulnerabilidades (CVE).
* **Firewall Evasion:** Técnicas de fragmentação e Decoy incorporadas.

* **Subdomain Finder:** Enumeração de subdomínios ocultos via brute-force.
* **Banner Grabbing:** Interação direta com portas para identificação real de serviços.
* **Rev-Shell Gen:** Gerador instantâneo de payloads para Reverse Shell (Bash, Python, Netcat).
